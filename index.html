Cole a URL do seu Cloudflare Worker abaixo. Depois disso os dados carregam automaticamente toda vez que abrir o arquivo.

1
Acessa workers.cloudflare.com → conta gratuita → Create Application → Worker → Edit Code
2
Cola o conteúdo do arquivo worker.js (enviado junto) → clica Deploy
3
No painel do Worker: Settings → Variables → Add variable
Nome: ASAAS_KEY · Valor: sua API Key · marca Encrypt → Save
4
Copia a URL do Worker (ex: dimplus.SEU-USUARIO.workers.dev) e cola abaixo
⚠ URL inválida — deve começar com https://
